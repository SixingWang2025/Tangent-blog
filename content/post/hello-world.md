## 关于本站

本博客使用 [Hugo](https://gohugo.io/) 静态网站生成器和 [Stack](https://github.com/CaiJimmy/hugo-theme-stack) 主题构建。

## 功能特性

- 支持明暗主题切换
- 全文搜索
- 文章分类与标签
- 响应式设计
- 代码高亮

## 代码示例

```python
def hello():
    print("Hello, Hugo!")
```

```go
package main

import "fmt"

    fmt.Println("Hello, Go!")
}
```
